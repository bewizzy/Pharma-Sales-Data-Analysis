# Pharma Sales Data Analysis (Power BI)

## Overview
This project analyzes pharmaceutical sales data using Power BI.  
The dashboard provides insights into sales performance across product categories, sales reps, channels, and time periods.

## Key Insights
- Total sales of ₦12bn across all years
- Analgesics and Antibiotics are top-performing product classes
- Hospital channel contributes the highest share of sales
- Sales performance varies significantly across representatives and teams

## Dashboard Features
- Sales distribution by product category
- Monthly and yearly sales trends
- Sales rep performance analysis
- Channel and sub-channel insights

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization
